# Campus 360 ERP System

A modern, high-fidelity Enterprise Resource Planning (ERP) system for educational institutions, built with React, Vite, and Material UI.

## 🚀 Key Features

- **Dynamic Student Dashboard**: Real-time attendance tracking, academic performance trends with Recharts, and automated Gazette generation.
- **Library Management**: SVG-driven modern interface for book tracking.
- **Smart Canteen**: Visual menu with category filtering and ordering system.
- **Glassmorphism UI**: High-fidelity design system using indigo/slate gradients and responsive layouts.
- **AICTE Integration**: Dedicated portal for regulatory compliance and institutional management.

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite
- **UI Components**: Material UI (MUI), Lucide React
- **Data Visualization**: Recharts
- **PDF Generation**: jsPDF
- **Date Management**: Day.js

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Ashu4495/ERP.git

# Navigate to project
cd ERP

# Install dependencies
npm install

# Start development server
npm run dev
```

## 🌐 Deployment

This project is configured to be served via a custom base path (`/ERP/`). To build for production:

```bash
npm run build
```
